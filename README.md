##  Key Features

-  Email/Password ও Google OAuth Authentication (Firebase)
-  User Profile Edit (Firebase updateProfile)
-  Protected Routes for Private Pages
-  Fully Responsive (Mobile, Tablet, Desktop)
-  Upcoming Events Section with JSON Data
-  Swiper Slider for Hero Section
-  Form Submission with Success Message
-  Forgot Password Functionality
-  Dynamic Title Based on Route
-  404 Not Found Page
-  Urban & Vibrant UI Design with AOS Animation
-  Hosted on Firebase Auth Domain Added

##  Pages Overview

- `/` — Home Page (Slider + Upcoming Events + Extra Sections)
- `/avourite_events` — Event Details (Protected)
- `/category` - category page
- `/login` — Login Page
- `/register` — Register Page
- `/profile` — My Profile Page (Protected)
- `/forgot-password` — Reset Password Page
- `/Booked` — Extra Protected Route Page
